Titulo do Projeto: Projeto Web ClassicGasosoline

Status do Projeto: em desenvolvimento

Tecnologias aplicadas:
Front-end: Linguagens HTML, CSS e JavaScript
Back-end: Java e MySQL ( integração com Spring MVC)

Objetivos do Software: Simulação de uma página web, estilo blog, com temática sobre carros antigos

alem de incluir e excluir sugestões.
